# OpenClaw Global Relay

A secure, multi-tenant WebSocket relay for OpenClaw Gateways. Enables access to the OpenClaw app and features without VPNs or tunnels.

## Project Structure
- `/relay`: The Azure Container Apps backend (Node.js).
- `/dashboard`: The Next.js web interface and landing page.
